# CardGuard

Is a ML that is trying to predict if the transation is fraud or not. The Data that have been used is from kaggle:
https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud/data

The features have been are only numerical input variables which are the result of a PCA transformation.

# Installation

To get the CSV use:
`curl -L -o ~/Downloads/creditcardfraud.ziphttps://www.kaggle.com/api/v1/datasets/download/mlg-ulb/creditcardfraud`
